# vuzix-blade

Experiments with speech and OCR / translate with the Vuzix Blade.

While the Vuzix Blade is 2018 technology, it does include offline capabilities for speech commands and optical character recognition in the AR view of the glasses.

In this repository will experiment with the overall setup.

Using Android Studio running on Apple Silicon / M1, which may pose issues, which I'll outline here. The main thing thus far is that virtual devices are not supported, so the device must be connected for debugging and testing.
